# 模块名称

Tool Bar

中文：

道具栏

技能栏

成长栏

---

# 模块定位

整局成长系统入口。

用于承载：

```text
道具
技能
Buff
流派
职业
成长路线
```

BuildArea决定当前战斗。

ToolBar决定后续成长方向。

---

# 玩家认知目标

玩家必须理解：

```text
这里是成长区

这里的东西会永久影响本局

这里的东西比征兵更稀有
```

---

# 区域位置

位于页面最底部。

ActionBar下方。

紧贴屏幕底边。

---

# 区域尺寸

width:720

height:120

---

# 视觉目标

玩家打开游戏时：

```text
征兵按钮最显眼

棋盘第二显眼

道具栏第三显眼
```

ToolBar不能抢征兵按钮视觉。
