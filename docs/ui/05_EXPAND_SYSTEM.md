# 05_EXPAND_SYSTEM.md

# 模块名称

Expand System

中文：

扩地区

开荒系统

土地扩张系统

---

# 模块定位

整个游戏长期成长核心。

决定：

```text
布阵空间

阵容规模

流派上限

后期强度
```

如果 BuildArea 决定战斗。

那么 ExpandArea 决定成长。

---

# 玩家认知目标

玩家进入游戏后必须理解：

```text
白色格子能放兵

红色格子不能放兵

红色格子未来能解锁

扩地后可以放更多兵
```

---

# 玩家决策目标

玩家需要不断思考：

```text
先扩左边？

先扩右边？

先扩上面？

还是继续征兵？
```

# 设计原则

玩家任何时候都应该看到：

```text
下一块地在哪

开地需要什么

开地后有什么收益
```

必须形成：

```text
征兵

↓

合成

↓

赚钱

↓

扩地

↓

更强阵容

↓

继续扩地
```

循环。
