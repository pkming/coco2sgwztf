# 03_PATH_SYSTEM.md

# 模块名称

Path System

中文：

敌军行军路线系统

---

# 模块定位

整个战斗区核心展示层。

玩家进入游戏后第一视觉焦点。

作用：

```text
展示敌人移动

展示战斗结果

展示箭矢

展示伤害

展示击杀反馈
```

路线本身不参与数值计算。

路线只负责：

```text
移动
展示
引导视线
```

---

# 玩家认知目标

玩家进入游戏3秒内必须理解：

```text
敌人从哪里来

敌人往哪里走

敌人要走到哪里

我的单位攻击谁
```

不允许出现：

```text
不知道怪物走哪

不知道出口在哪

不知道路线方向
```
