# 《字战三国》占位资源与 image2 生成说明

当前 P0 原型全部使用 Cocos Graphics 和文字占位，不依赖真实图片。后续需要替换成正式美术时，按下面说明生成或设计。

## 视觉总方向

- 画面比例：移动端竖屏，推荐 720×1280 或 1080×1920。
- 风格：三国古籍感 + 清晰手游 UI，不要写实复杂背景。
- 主视觉重点：汉字单位本身要最大、最清楚，玩家一眼能认出“刀、骑、弓、枪”。
- 色彩：米黄宣纸底、深棕路径、红/蓝/绿/金区分单位。
- 禁忌：不要让插画抢过文字，不要给每个字加太多装饰导致不可读。

## 四个基础单位视觉规则

### 刀
- 颜色建议：赤红。
- 字感：厚重、锋利、单体爆发。
- 动效：重击、短距离挥砍、命中闪白。

### 骑
- 颜色建议：金橙。
- 字感：扫荡、冲击、范围攻击。
- 动效：圆形范围波纹或马蹄冲击。

### 弓
- 颜色建议：蓝色。
- 字感：远程、精准、覆盖距离。
- 动效：字后拉，箭矢飞出，远处命中。

### 枪
- 颜色建议：绿色。
- 字感：直线、穿刺、队列伤害。
- 动效：一条直线刺击，穿过多个敌人。

## 当前原型如何替换图片

当前 `assets/scripts/core/GameBootstrap.ts` 使用 Graphics 绘制占位块。等图片资源准备好后：

1. 在 Cocos Creator 里导入 PNG。
2. 给对应节点添加 Sprite。
3. 用图片 SpriteFrame 替换 Graphics 背景。
4. 保留 Label 显示汉字和等级。

优先替换顺序：

```text
unit_tile_base.png
↓
board_bg.png
↓
path_tile.png
↓
enemy_normal.png
↓
ui_button.png
```

### 页面设计目标

- 一眼看出是三国题材。
- 战场区域偏暗，方便突出敌人、路线和攻击效果。
- 棋盘区域比战场更规整，像“背包/棋盘”。
- 汉字单位要比普通图标更醒目。
- 按钮要像手游底部操作栏，方便拇指点击。

```text

```
