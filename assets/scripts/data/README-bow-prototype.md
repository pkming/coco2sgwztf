# 弓箭职业首发玩法原型 V0.1

## 目标

先落地固定站位弓箭 build 的配置层，验证：

- 职业固定技能树是否能形成三条路线。
- 装备词条是否能强化技能等级和 build 方向。
- 怪物血量差异和数量压力是否能支撑首个战斗原型。

## 已新增配置

- `BowSkillTreeConfig.ts`：弓箭职业技能树，P0 先做 5 个技能。
- `EquipmentAffixConfig.ts`：4 个装备部位和 9 类核心词条。
- `BowMonsterPressureConfig.ts`：小怪、厚血怪、精英怪和 5 波首关压力配置。
- `BowPrototypeConfig.ts`：首发原型汇总入口，供后续战斗或 UI 读取。

## P0 技能

- 基础箭
- 连射
- 瞄准射击
- 穿透箭
- 元素附着

## P0 装备部位

- 武器
- 项链
- 戒指
- 手套

## 下一步接入建议

1. 新建弓箭战斗 Bootstrap，读取 `BOW_PROTOTYPE_V0_1`。
2. 先实现基础自动射击、攻速、穿透和元素伤害四类效果。
3. 用 `P0_BOW_WAVES` 替换旧 P0 原型里的怪物生成规则。
4. 再接装备掉落与技能点分配 UI。
